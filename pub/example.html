Specify Demo

Getting Started

This is a text checker library, the developer will design rules that clients need to follow.

There are several pre-defined warning rules:

missing-front
missing-back
missing
lengthAtLeast
lengthAtMost

If non of the predefined rule is useful to you, you can define your own rule with regex and set the type of rule to:

other

There are several warning animation:

rotate-c (rotate counter clockwise)
rotate-cc (rotate counterclockwise)
shake
bounce
float
warning-colorChange

Example

Enter "a", click check, you will encounter the first rule, animation is shaking and there is a gif-like toturial teaching you how to fix it

Enter "UofT_a", click check, you will encounter the second rule, animation is shaking, follow the rule and correct it

Enter "UofT_aaaaaaa", click check, you will encounter the third rule, animation is warning-colorChange and there is a gif-like toturial teaching you how to fix it

Enter "UofT_aaaaaaa@gaming", click check, you will encounter the forth rule, animation is rotate the color couterclockwise and there is a gif-like toturial teaching you how to fix it

Enter "UofT_aaaaaaacura@gaming", click check, you will encounter the fifth rule, which is a customed rule with the developer put in his own regex to create the rule, animation is rotate the color clockwise

Enter "UofT_aaaaaaaaaaaaaaaaaaaaaaaa9aaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaaacura@gaming", click check, you will encounter the sixth rule, the animation is float

Enter "UofT_aaaaaaacura9@gaming", click check, you pass all rules, congratulations!
